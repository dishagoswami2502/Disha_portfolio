# Disha's Professional Portfolio

A stunning, dark-mode portfolio website for Disha — Regional Manager & Robotics Program Manager.

## 🌐 Live Site
Deployed on GitHub Pages: [View Portfolio](https://dishagoswami2502.github.io/Disha_portfolio/)

## ✨ Features
- 🎨 Dark mode with glassmorphism design
- 🤖 Animated typing text
- 🌟 Custom cursor with smooth following
- 📱 Fully responsive (mobile-friendly)
- ✨ Floating particle system
- 🖼️ Interactive gallery with lightbox
- 📊 Counter animations
- ⚡ Scroll-triggered fade-in animations
- 🖱️ Parallax mouse effects

## 📁 Structure
```
Disha_portfolio/
├── index.html       — Main portfolio page
├── style.css        — All styles (dark mode, animations)
├── script.js        — Interactivity & animations
├── Disha_resume.pdf — Downloadable resume
└── *.jpeg           — Portfolio images
```

## 🚀 Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Outfit, Space Grotesk)
- IntersectionObserver API for scroll animations
- CSS custom properties for theming
